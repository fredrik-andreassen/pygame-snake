### Avhengigheter
- [`pygame`](https://www.pygame.org/wiki/GettingStarted)

### Kjør spillet
#### Gammel utgave
Standard vindusstørrelse: `python old/main.py`

Egendefinert vindusstørrelse: `python old/main.py 1920 1080`

#### Ny utgave
`python main.py`

Feilsøkingsmodus: `python main.py debug [FPS]`
